## 公开网址导航

**用户价值**：作为访客，我希望快速浏览和搜索保存的网站，以便在需要时直接跳转访问。

**验收标准**：

- Given 数据库存在网站记录，When 访客打开首页，Then 页面展示网站名称、URL 和备注。
- Given 访客点击网站链接，When 浏览器处理外链，Then 目标网站在新标签页打开且原页面保持不变。
- Given 访客输入关键词，When 网站名称、URL 或备注匹配，Then 列表只展示匹配记录。
- Given 用户切换语言，When 选择简体中文或 English，Then 界面文案切换且网站数据保持不变。
- Given 数据库暂无记录，When 访客打开首页，Then 页面展示空状态。

**安全关卡**：

- 认证：公开读取不需要登录。
- 输入校验：搜索只在客户端过滤，不执行危险 HTML。
- XSS 防护：网站名称和备注通过 React 文本节点渲染。
- 外链安全：链接使用 `target="_blank"` 和 `rel="noopener noreferrer"`。

**不在范围内**：公开评论、用户提交网址。
