## 网站记录维护

**用户价值**：作为管理员，我希望维护网站名称、网址和备注，以便公开首页持续展示最新收藏。

**验收标准**：

- Given 管理员已登录，When 创建合法网站记录，Then 记录保存到数据库并出现在后台列表和公开首页。
- Given 管理员已登录，When 编辑网站名称、URL、备注或排序值，Then 系统保存并展示最新内容。
- Given 管理员已登录，When 删除网站并确认，Then 记录从公开首页消失。
- Given 未登录用户，When 调用创建、编辑或删除接口，Then 系统返回 401 且不修改数据库。
- Given 输入空名称、非法 URL 或超长备注，When 提交表单，Then 前后端都返回字段级错误。

**安全关卡**：

- 需要认证：创建、更新、删除需要登录。
- 输入校验：名称、URL、备注、排序值使用 Zod 校验。
- 数据库操作：使用 Prisma ORM。
- XSS 防护：React 默认转义用户输入，不使用 `dangerouslySetInnerHTML`。
- CSRF 防护：写接口检查同源请求，Cookie 使用 SameSite=Lax。

**不在范围内**：批量导入、标签分类、网站可用性监控。
