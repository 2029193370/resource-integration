# 业务逻辑文档

> 版本：v1.0 | 创建日期：2026-05-12 | 项目：网址收藏导航

## 1. 项目概述

### 1.1 项目背景

项目用于长期保存和维护常用网站，避免因为浏览器书签、聊天记录或临时文档分散导致网址找不到。访问者可以在公开首页浏览网站名称、网址和备注，并点击网址直接跳转；管理员通过后台维护这些记录。

### 1.2 项目目标

- **主要目标**：上线一个可部署到 Vercel 的全栈网站，支持公开浏览网站列表和管理员后台 CRUD。
- **次要目标**：提供简体中文和英文双语言界面，配置通过 `.env` 抽离，数据库使用 Neon PostgreSQL。
- **非目标（Out of Scope）**：多用户协作、复杂权限体系、浏览器插件、网站可用性监控、批量爬虫导入。

### 1.3 目标用户

| 用户角色 | 描述 | 核心诉求 |
|---------|------|---------|
| 访客 | 打开网站查找已保存的网址 | 快速搜索、查看备注、点击跳转 |
| 管理员 | 维护网址数据的人 | 登录后台后新增、编辑、删除网站记录 |

## 2. 核心业务流程

### 2.1 公开浏览流程

```text
访客进入首页 -> 系统加载已发布网站列表 -> 访客搜索或切换语言 -> 点击网址 -> 新标签页跳转目标网站
```

### 2.2 管理维护流程

```text
管理员进入 /admin/login -> 输入账号密码 -> 登录成功进入后台 -> 创建/编辑/删除网站 -> 数据保存到 Neon PostgreSQL -> 首页实时展示最新记录
```

### 2.3 异常处理

- 网站列表为空时展示空状态，并引导管理员登录添加第一条记录。
- 目标网址必须是合法 `http` 或 `https` URL，非法 URL 在前后端都拒绝保存。
- 管理员未登录访问后台页面时跳转登录页。
- 删除网站前必须二次确认，避免误删。

### 3.2 功能详细描述

#### 网站列表展示

- **描述**：公开首页加载数据库中的网站记录，以卡片形式展示。
- **输入**：无必填输入，可选搜索关键词和语言。
- **输出**：网站名称、可点击 URL、备注、更新时间。
- **业务规则**：仅展示未删除记录；默认按 `sortOrder` 升序、创建时间降序排序。
- **异常场景**：数据库连接失败时显示友好错误，不暴露内部堆栈。

#### 管理员登录

- **描述**：管理员通过 `/admin/login` 登录后台。
- **输入**：用户名、密码。
- **输出**：登录成功后写入安全会话 Cookie。
- **业务规则**：密码使用 bcrypt 校验；Cookie 使用 `HttpOnly`、`Secure`、`SameSite=Lax`。
- **异常场景**：账号或密码错误返回统一错误文案，避免枚举账号。

#### 网站 CRUD

- **描述**：管理员维护网站记录。
- **输入**：名称、URL、备注、排序值。
- **输出**：保存后的记录。
- **业务规则**：名称必填且不超过 80 字符；URL 必须为 `http` 或 `https`；备注不超过 500 字符。
- **异常场景**：未登录返回 401；记录不存在返回 404；校验失败返回字段级错误。

## 4. 非功能需求

| 需求类型 | 指标 | 说明 |
|---------|------|------|
| 性能 | 首页首屏加载 < 2s，API 响应 < 500ms | 使用 Vercel Edge/CDN 和数据库索引 |
| 可用性 | 部署平台可用性依赖 Vercel 与 Neon | 页面需有加载、错误、空状态 |
| 安全性 | 后台接口全部鉴权，敏感配置只读 `.env` | 不提交真实密钥 |
| 兼容性 | 支持 Chrome、Edge、Safari、Firefox 最新两个版本 | 响应式适配手机和桌面 |
| 可维护性 | Prisma schema + migration 管理数据库 | README 写清本地和部署步骤 |

## 6. 业务规则

| 规则编号 | 规则描述 | 触发场景 |
|---------|---------|---------|
| BR-001 | 网站 URL 必须是 `http` 或 `https` 开头的合法地址 | 新增或编辑网站 |
| BR-002 | 后台写操作必须登录管理员 | 创建、更新、删除网站 |
| BR-003 | 公开首页外链必须新标签页打开并设置 `rel="noopener noreferrer"` | 点击网站链接 |
| BR-004 | 管理员密码不得明文存储 | 初始化管理员或修改密码 |
| BR-005 | `.env` 存储数据库、认证密钥和初始管理员配置 | 本地开发与生产部署 |
